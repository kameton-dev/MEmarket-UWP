**Millennium Market (MEmarket)** — это современный клиент магазина приложений для Windows 10 Mobile, работающий на основе простых JSON-репозиториев, которые могут создавать абсолютно все, тем самым распространяя больше приложений.

> [!IMPORTANT]
> Клиенты и сама архитектура MEmarket сейчас в стадии **активной разработки**. Будьте готовы к частым обновлениям, изменений архитектуры репозиторий и возможными багами

---

## О данном проекте
Этот проект делает упор на **открытый исходный код и поддержку сообщества**. **Каждый** может создать свой собственный репозиторий и разместить его на своём сервере либо захостить через GitHub Pages (как пример). Приложение **не** несет никакой корпоративной цели и **не** зависит от одного сервера, это означает что проект будет жив вместе с сообществом.

## Особенности:

- **Минимальные зависимости** — простая архитектура на C# / UWP.
- **Репозитории** — поддержка нескольких репозиториев одновременно и их управление.
- **Кроссплатформенность** — поддержка приложений под Windows Phone 8.1 и Windows 10 Mobile.
- **Простота** — репозитории хранятся в простом JSON-формате, легко генерируются и обновляются.

## Быстрый старт

### Предварительные требования 
- Visual Studio 2017 (Поддерживает развертывание UWP приложений на W10M).
- Устройство на Windows 10 Mobile (мин. сборка - 1703).

### 1. Клонирование репозитория
```bash
git clone --recurse-submodules https://github.com/monogram-android/monogram.git
```
### 2. Сборка проекта
- Откройте решение MEmarket-UWP.sln в Visual Studio
- Выберите целевую платформу (часто ARM)
- Собрать и запустить приложение через VS
