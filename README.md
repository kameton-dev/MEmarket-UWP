**MEmarket UWP** — это клиент магазин приложений Millennium Market для Windows 10 Mobile, работающий на основе простых JSON-репозиториев, которые могут создавать абсолютно все, тем самым распространяя больше приложений.

> [!IMPORTANT]
> Приложение сейчас находится в стадии **активной разработки**. Будьте готовы к частым обновлениям и возможным ошибкам

---

## О Millennium Market

**Millennium Market (MEmarket)** — это официальный репозиторий проверенных и самых необходимых приложений (MEmarket-Basic), а также комплект инструментов для настройки и комфортной работы с магазином. Проект был создан и по сей день развивается одним энтузиастом благодаря поддержке нашего небольшого, но дружного сообщества. Главная цель MEmarket — доказать, что такой подход не обязательно должен быть сложным и доступным только «для гиков». Мы делаем платформу, которая максимально дружелюбна и понятна для каждого обычного пользователя. 

## Особенности:

- **Минимальные зависимости** — простая архитектура на C# / UWP.
- **Отслеживание обновлений** — полноценная функция обновлений пакетов, отслеживание а также их управление.
- **Репозитории** — поддержка нескольких репозиториев одновременно и их управление.
- **Кроссплатформенность** — поддержка приложений под Windows Phone 8.1 и Windows 10 Mobile.
- **Простота** — репозитории хранятся в простом JSON-формате, легко генерируются и обновляются.

## Быстрый старт

### Предварительные требования 
- Visual Studio 2017 (Поддерживает развертывание UWP приложений на W10M).
- Устройство на Windows 10 Mobile c мин. сборкой 1607.

### 1. Клонирование репозитория
```bash
git clone https://github.com/kameton-dev/MEmarket-UWP.git
```
### 2. Сборка проекта
- Откройте решение MEmarket-UWP.sln в Visual Studio
- Выберите целевую платформу (часто ARM)
- Собрать и запустить приложение через VS
